/wLib - w라이브러리(공통모듈, LOG, CONF)
IDisposable 인터페이스 상속) 가비지 수집기는 해당 개체가 더 이상 사용되지 않을 때 관리되는 개체에 할당된 메모리를 자동으로 해제합니다.
1. DB - MYSQL 각 공통으로 사용되는 DDL 및 DCL 기록 (DCL은 Create만 사용)
2. Protocol - 각 컨트롤 보드(펌웨어)와 약속된 통신규약 기록. 본문에선 간단한 함수로 작동 할 수 있게 정리
3. Conf - 각 프로그램이 실행될 때 필요한 Setting 방식을 etc/ini를 통해 관리될 수 있도록 기록. Mutex를 사용해 먼저 사용중인 프로그램이 사용 후 사용할 수 있도록 정리
4. UI - 프로그램마다 공통으로 사용되는 UI 기록. (인증때문에 우회하기 위해도 사용.
